# Garbagotchi
Take care of your virtual pet by sorting waste into the correct bins!

## Install
Requirements:
- python
- pip

    pip install pipenv
    pipenv install

## Localhost
    pipenv run flask run

The app will be live at [http://127.0.0.1:5000/](http://127.0.0.1:5000/)

Specify a port:
    pipenv run flask run --port \<port\>
